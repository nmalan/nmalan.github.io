# nmalan.github.io

Personal academic website for Neil Malan, Physical Oceanographer.

## Site traffic

Visitor stats (page views, countries, referrers) are tracked via [GoatCounter](https://goatcounter.com). Check the dashboard at [nmalan.goatcounter.com](https://nmalan.goatcounter.com).

## Google Scholar stats

Citation counts are updated automatically every Monday at 03:00 UTC via the `update-scholar-stats` GitHub Actions workflow. The workflow fetches data using the `scholarly` Python package and commits any changes to `index.html`.
