# nmalan.github.io

Personal academic website for Neil Malan, Physical Oceanographer.

## Site traffic

Visitor stats (page views, countries, referrers) are tracked via [GoatCounter](https://goatcounter.com). Check the dashboard at [nmalan.goatcounter.com](https://nmalan.goatcounter.com).

## Google Scholar stats

Citation counts (lines 818 and 822 of `index.html`) must be updated manually by editing `index.html` directly. Check your current stats at [Google Scholar](https://scholar.google.com) and update the `Citations` and `h-index` values as needed.
